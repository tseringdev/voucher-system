# Voucher Management System

A web-based application to create, manage, and track financial vouchers in a simple and organized way.

## 🚀 Features
- Add income and expense vouchers
- Form validation
- Dynamic data handling using JavaScript
- Organized interface

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript
